# Glossário — Dev Academy

## Termos do Projeto

| Termo | Definição |
|-------|-----------|
| **Conteúdo** | Nível superior da hierarquia (ex: "JavaScript", "React"). Agrupa tópicos relacionados. |
| **Tópico** | Nível intermediário (ex: "Funções", "Hooks"). Agrupa aulas sobre um assunto específico. |
| **Aula** | Nível inferior. Documento `.mdx` com conteúdo educacional e exercício. |
| **Exercício** | Prática ao final de cada aula. Editor de código com testes de validação. |
| **XP** | Pontos de experiência ganhos ao concluir aulas e exercícios. |
| **Nível** | Classificação baseada no XP total acumulado. |
| **Conquista** | Badge desbloqueada ao atingir critérios específicos. |
| **Frontmatter** | Metadados YAML no início de arquivos `.mdx` (id, título, XP, exercício). |
| **Sidebar** | Painel lateral de navegação com conteúdos, tópicos e aulas. |

## Termos Técnicos

| Termo | Definição |
|-------|-----------|
| **Next.js** | Framework React para aplicações web com SSR/SSG. |
| **App Router** | Sistema de roteamento do Next.js 14+ baseado em pastas. |
| **MDX** | Formato que combina Markdown com componentes React. |
| **Tailwind CSS** | Framework CSS utilitário para estilização rápida. |
| **shadcn/ui** | Biblioteca de componentes React acessíveis e customizáveis. |
| **Zustand** | Biblioteca leve de gerenciamento de estado para React. |
| **Framer Motion** | Biblioteca de animações para React. |
| **Monaco Editor** | Editor de código usado no VS Code. |
| **Sandpack** | Ambiente de execução de código no navegador. |
| **Prisma** | ORM para banco de dados com tipagem TypeScript. |
| **SQLite** | Banco de dados leve baseado em arquivo. |
| **NextAuth.js** | Biblioteca de autenticação para Next.js. |

## Termos de Design

| Termo | Definição |
|-------|-----------|
| **Tema Escuro** | Paleta de cores escuras, padrão para desenvolvedores. |
| **Glassmorphism** | Efeito visual com transparência e blur (vidro fosco). |
| **Microinteração** | Pequena animação em resposta a ação do usuário. |
| **WCAG** | Diretrizes de Acessibilidade para Conteúdo Web. |
| **Responsivo** | Layout que se adapta a diferentes tamanhos de tela. |

## Siglas

| Sigla | Significado |
|-------|-------------|
| **SSR** | Server-Side Rendering |
| **SSG** | Static Site Generation |
| **MVP** | Minimum Viable Product (Produto Mínimo Viável) |
| **UAT** | User Acceptance Testing (Teste de Aceitação do Usuário) |
| **ORM** | Object-Relational Mapping |
| **API** | Application Programming Interface |
| **UI** | User Interface |
| **UX** | User Experience |
