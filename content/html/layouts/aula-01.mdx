## Div

`<div>` é um container genérico para agrupar elementos:

```html
<div class="card">
  <h2>Título</h2>
  <p>Conteúdo</p>
</div>
```

### Quando usar `<div>`

- Agrupar elementos relacionados
- Aplicar estilos a múltiplos elementos
- Criar seções da página
- Container para layout

## Span

`<span>` é inline - para estilizar partes de texto:

```html
<p>Texto com <span class="destaque">palavra importante</span> no meio.</p>
```

### Diferença: Div vs Span

| Característica | `<div>` | `<span>` |
|----------------|---------|----------|
| Display | Block (ocupa linha inteira) | Inline (no fluxo do texto) |
| Uso | Containers, seções | Estilizar texto |
| Aninhamento | Pode conter qualquer coisa | Só conteúdo inline |

## Exemplo: Card Component

```html
<div class="card">
  <div class="card-header">
    <h3>Título do Card</h3>
  </div>
  <div class="card-body">
    <p>Conteúdo do card aqui.</p>
  </div>
  <div class="card-footer">
    <button>Ação</button>
  </div>
</div>
```

## Exemplo: Alert com Span

```html
<div class="alert">
  <span class="icon">⚠️</span>
  <span class="message">Atenção! Verifique os dados.</span>
</div>
```

## Boas Práticas

1. Use `<div>` para containers e seções
2. Use `<span>` para estilizar texto inline
3. Prefira tags semânticas quando possível (`<header>`, `<nav>`, etc.)
4. Use classes descritivas (`.card`, `.alert`, `.sidebar`)
