## Introdução

Acessibilidade garante que pessoas com deficiência possam usar seu site. HTML semântico é o primeiro passo.

## Conceito Explicado

```html
<!-- Imagem com alt descritivo -->
<img src="foto.jpg" alt="Pessoa sorrindo em um parque">

<!-- Botão com aria-label para leitores de tela -->
<button aria-label="Fechar modal">X</button>

<!-- Input com label associado -->
<label for="email">Email:</label>
<input type="email" id="email" name="email">

<!-- Navegação com landmarks -->
<nav aria-label="Menu principal">
    <ul>
        <li><a href="/">Início</a></li>
    </ul>
</nav>
```

## Exemplo Visual

**Sem acessibilidade:**
```html
<img src="logo.png">
<button>X</button>
<input type="text">
```

**Com acessibilidade:**
```html
<img src="logo.png" alt="Logo da empresa">
<button aria-label="Fechar">X</button>
<label for="nome">Nome:</label>
<input type="text" id="nome">
```

Leitores de tela agora entendem cada elemento.

## Boas Práticas

- ✅ Sempre use `alt` descritivo em imagens
- ✅ Associe `<label>` a inputs com `for`/`id`
- ✅ Use `aria-label` em botões sem texto visível
- ✅ Use landmarks (`<nav>`, `<main>`, `<header>`)
- ❌ Não use `alt=""` em imagens informativas
- ❌ Não use `<div>` ou `<span>` como botão

## Atividade Final

Corrija a página fornecida:

2. Adicione `aria-label` no botão
3. Adicione `<label>` associado ao input
