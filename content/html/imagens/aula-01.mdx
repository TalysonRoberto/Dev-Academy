## Imagens

Use `<img>` para exibir imagens:

```html
<img src="foto.jpg" alt="Minha foto" width="300" height="200">
```

### Atributos Importantes

| Atributo | Obrigatório | Função |
|----------|-------------|--------|
| `src` | Sim | Caminho da imagem |
| `alt` | Sim | Texto alternativo (acessibilidade) |
| `width` | Não | Largura em pixels |
| `height` | Não | Altura em pixels |
| `loading` | Não | `lazy` para carregamento preguiçoso |

### Imagem Responsiva

```html
<img src="foto.jpg" alt="Descrição" style="max-width: 100%; height: auto;">
```

## Vídeo

```html
<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  Seu navegador não suporta vídeo.
</video>
```

### Atributos de Vídeo

| Atributo | Função |
|----------|--------|
| `controls` | Mostra controles (play, pause, volume) |
| `autoplay` | Inicia automaticamente |
| `loop` | Repete ao terminar |
| `muted` | Inicia sem som |

## Áudio

```html
<audio controls>
  <source src="musica.mp3" type="audio/mpeg">
  Seu navegador não suporta áudio.
</audio>
```

## Iframe

Incorpora conteúdo de outros sites:

```html
<iframe 
  width="560" 
  height="315" 
  src="https://www.youtube.com/embed/VIDEO_ID" 
  title="YouTube video"
  frameborder="0" 
  allowfullscreen>
</iframe>
```

## Boas Práticas

1. **Sempre use `alt`** em imagens (acessibilidade)
2. **Defina width/height** para evitar layout shift
3. **Use `loading="lazy"`** para imagens abaixo da dobra
4. **Otimize imagens** antes de usar (comprima, redimensione)

- `width` de 300
