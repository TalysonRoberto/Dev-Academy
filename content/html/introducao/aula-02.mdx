## Elementos

Um **elemento** HTML é composto por:
- **Tag de abertura**: `<tagname>`
- **Conteúdo**: O texto ou outros elementos dentro
- **Tag de fechamento**: `</tagname>`

```html
<p>Este é um parágrafo.</p>
```

## Atributos

**Atributos** fornecem informações adicionais sobre um elemento. Eles ficam na tag de abertura:

```html
<a href="https://google.com" target="_blank">Google</a>
```

### Atributos Comuns

| Atributo | Descrição | Exemplo |
|----------|-----------|---------|
| `href` | URL do link | `href="https://google.com"` |
| `src` | Caminho da imagem | `src="foto.jpg"` |
| `alt` | Texto alternativo | `alt="Descrição da foto"` |
| `id` | Identificador único | `id="meu-id"` |
| `class` | Classe CSS | `class="destaque"` |
| `target` | Onde abrir link | `target="_blank"` |

## Elementos com Auto-Fechamento

Alguns elementos não têm tag de fechamento:

```html
<img src="foto.jpg" alt="Minha foto">
<br>
<hr>
<input type="text">
```

Crie um link que:
- Aponte para `https://google.com`
