## Introdução

**Elementos** são os blocos de construção do HTML. **Atributos** fornecem informações adicionais sobre um elemento.

## Conceito Explicado

```html
<tagname atributo="valor">Conteúdo</tagname>
```

### Exemplo com atributos:

```html
<!-- Link com href e target -->
<a href="https://google.com" target="_blank">Google</a>

<!-- Imagem com src e alt -->
<img src="foto.jpg" alt="Minha foto">

<!-- Input com type e placeholder -->
<input type="text" placeholder="Seu nome">
```

## Exemplo Visual

**Sem atributos:**
```html
<a>Google</a>          <!-- Sem href → não é um link -->
<img>                   <!-- Sem src → sem imagem -->
<input>                 <!-- Sem type → padrão text -->
```

**Com atributos:**
```html
<a href="https://google.com" target="_blank">Google</a>
<img src="foto.jpg" alt="Descrição da foto">
<input type="email" placeholder="seu@email.com">
```

Atributos transformam tags genéricas em elementos funcionais.

## Boas Práticas

- ✅ Sempre use `alt` em `<img>` (acessibilidade)
- ✅ Use `type` correto em `<input>` (email, number, etc.)
- ✅ Use `target="_blank"` com `rel="noopener noreferrer"` em links externos
- ❌ Não use atributos sem valores
- ❌ Não coloque espaços entre o nome do atributo e o `=`

## Atividade Final

1. Um link `<a>` com `href` para o Google
2. Uma imagem `<img>` com `src` e `alt`
3. Um input `<input>` com `type` e `placeholder`
