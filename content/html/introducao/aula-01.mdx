## Introdução

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ela define a **estrutura** e o **conteúdo** de uma página.

## Conceito Explicado

Todo documento HTML é uma árvore de **elementos**. Cada elemento é definido por uma tag:

```html
<tagname atributo="valor">Conteúdo</tagname>
```

A estrutura básica de qualquer página:

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é meu primeiro site.</p>
</body>
</html>
```

## Exemplo Visual

**Sem HTML** (texto puro):
```
Olá, Mundo!
Este é meu primeiro site.
```

**Com HTML** (página estruturada):
```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <title>Meu Site</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é meu primeiro site.</p>
</body>
</html>
```

O HTML transforma texto cru em uma página com hierarquia, semântica e estilos padrão do navegador.

## Boas Práticas

- ✅ Sempre declare `<!DOCTYPE html>` no início
- ✅ Use `lang="pt-BR"` no `<html>` para acessibilidade
- ✅ Inclua `<meta charset="UTF-8">` para caracteres especiais
- ❌ Não esqueça as tags de fechamento (`</html>`, `</body>`)
- ❌ Não coloque conteúdo visível dentro de `<head>`
