## Introdução

Tabelas organizam dados em linhas e colunas. Use apenas para dados tabulares, nunca para layout.

## Conceito Explicado

```html
<table>
    <thead>
        <tr>
            <th>Nome</th>
            <th>Idade</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Ana</td>
            <td>25</td>
        </tr>
        <tr>
            <td>João</td>
            <td>30</td>
        </tr>
    </tbody>
</table>
```

## Boas Práticas

- ✅ Use `<thead>` para cabeçalho
- ✅ Use `<tbody>` para corpo
- ✅ Use `<th>` para células de cabeçalho
- ❌ Não use tabelas para layout de página
- ❌ Não aninhe tabelas desnecessariamente
