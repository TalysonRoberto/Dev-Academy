## Introdução

Este é o projeto final de HTML. Você vai usar tudo que aprendeu para criar uma landing page completa e semântica.

## Conceito Explicado

Uma landing page típica tem:

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Landing Page</title>
</head>
<body>
    <header>
        <nav><!-- Menu --></nav>
    </header>
    <main>
        <section id="hero"><!-- Hero --></section>
        <section id="sobre"><!-- Sobre --></section>
        <section id="contato"><!-- Contato --></section>
    </main>
    <footer><!-- Rodapé --></footer>
</body>
</html>
```

## Exemplo Visual

**Estrutura da landing page:**
```
┌─────────────────────────────────┐
│ HEADER + NAV                    │
├─────────────────────────────────┤
│ SECTION: Hero (título + CTA)    │
├─────────────────────────────────┤
│ SECTION: Sobre (texto + img)    │
├─────────────────────────────────┤
│ SECTION: Contato (formulário)   │
├─────────────────────────────────┤
│ FOOTER (links + copyright)      │
└─────────────────────────────────┘
```

## Boas Práticas

- ✅ Use tags semânticas para toda a estrutura
- ✅ Inclua `meta viewport` para responsividade
- ✅ Use IDs em sections para navegação por âncoras
- ❌ Não use `<div>` quando existe tag semântica
- ❌ Não esqueça o `<footer>`

## Atividade Final

Construa uma landing page com:
1. `<header>` com `<nav>` contendo links
2. `<main>` com 3 `<section>` (hero, sobre, contato)
3. `<footer>` com informações de rodapé
4. Use apenas tags semânticas (sem `<div>`)
