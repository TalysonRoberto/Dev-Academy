## Introdução

HTML5 trouxe muitos tipos de input específicos. Cada tipo ativa comportamentos nativos do navegador (calendário, validação, teclado otimizado).

## Conceito Explicado

```html
<input type="text">       <!-- Texto livre -->
<input type="email">      <!-- Valida email automaticamente -->
<input type="password">   <!-- Oculta o texto -->
<input type="number">     <!-- Apenas números -->
<input type="date">       <!-- Seletor de data -->
<input type="tel">        <!-- Teclado numérico em mobile -->
<input type="url">        <!-- Valida URL -->
<input type="range">      <!-- Slider -->
<input type="color">      <!-- Seletor de cor -->
<textarea></textarea>     <!-- Texto multilinha -->
```

## Exemplo Visual

```html
<form>
    <input type="text" placeholder="Nome">
    <input type="email" placeholder="Email">
    <input type="number" min="0" max="120" placeholder="Idade">
    <input type="date">
    <input type="tel" placeholder="Telefone">
    <textarea placeholder="Mensagem"></textarea>
</form>
```

**Preview:** Cada input renderiza diferente — date mostra calendário, number mostra botões +/-.

## Boas Práticas

- ✅ Use `type="email"` para emails (validação automática)
- ✅ Use `type="number"` com `min`/`max` para limites
- ✅ Use `type="tel"` em mobile (teclado numérico)
- ❌ Não use `type="text"` para tudo
- ❌ Não esqueça `required` em campos obrigatórios

## Atividade Final

Crie um formulário de contato com:

2. Email (`type="email"`)
3. Idade (`type="number"`)
4. Data preferida (`type="date"`)
