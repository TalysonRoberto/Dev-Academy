## Introdução

Formulários coletam dados do usuário. São essenciais para cadastros, logins, buscas e qualquer interação que envolva envio de dados.

## Conceito Explicado

```html
<form action="/cadastro" method="POST">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <button type="submit">Cadastrar</button>
</form>
```

### Elementos de formulário:

| Elemento | Função |
|----------|--------|
| `<form>` | Container do formulário |
| `<label>` | Rótulo do campo |
| `<input>` | Campo de entrada |
| `<select>` | Lista suspensa |
| `<textarea>` | Campo de texto multilinha |
| `<button>` | Botão de ação |

## Exemplo Visual

```html
<form>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required placeholder="Seu nome">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required placeholder="seu@email.com">
    
    <label for="senha">Senha:</label>
    <input type="password" id="senha" name="senha" required minlength="6">
    
    <button type="submit">Enviar</button>
</form>
```

## Boas Práticas

- ✅ Sempre use `<label>` com `for` correspondente ao `id` do input
- ✅ Use `required` para campos obrigatórios
- ✅ Use `type` correto (email, password, number) para validação automática
- ❌ Não esqueça o `name` nos inputs (necessário para envio)
- ❌ Não use `placeholder` como substituto de `<label>`

## Atividade Final

1. Campo de nome (`type="text"`, `required`)
2. Campo de email (`type="email"`, `required`)
3. Campo de senha (`type="password"`, `required`)
4. Campo de data de nascimento (`type="date"`)
5. Botão de submit
