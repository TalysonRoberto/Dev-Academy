## Introdução

Títulos e parágrafos são os elementos mais básicos do HTML. Eles definem a hierarquia e o fluxo do texto.

## Conceito Explicado

HTML tem 6 níveis de títulos:

```html
<h1>Título Principal (maior)</h1>
<h2>Subtítulo</h2>
<h3>Seção</h3>
<h4>Subseção</h4>
<h5>Detalhe</h5>
<h6>Menor nível</h6>
```

Parágrafos são definidos com `<p>`:

```html
<p>Este é um parágrafo de texto.</p>
<p>Este é outro parágrafo separado.</p>
```

## Exemplo Visual

```html
<h1>Meu Blog</h1>
<h2>Primeiro Post</h2>
<p>Este é o conteúdo do meu primeiro post.</p>
<p>Espero que gostem!</p>

<h2>Segundo Post</h2>
<p>Outro post interessante.</p>
```

**Preview renderizado:**
- "Meu Blog" aparece em tamanho grande (h1)
- "Primeiro Post" em tamanho médio (h2)
- Os parágrafos com espaçamento automático

## Boas Práticas

- ✅ Use apenas **um `<h1>`** por página
- ✅ Não pule níveis (não vá de `h1` para `h4`)
- ✅ Use `<p>` para textos, não `<br>` repetidos
- ❌ Não use títulos apenas para deixar texto grande (use CSS)

## Atividade Final

Monte uma página com:
1. Um título `h1` principal
2. Dois subtítulos `h2`
3. Um `h3` dentro de um dos `h2`
4. Pelo menos 2 parágrafos `<p>`
