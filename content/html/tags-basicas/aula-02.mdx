## Listas

### Lista Não-Ordenada (com bullets)

```html
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
```

### Lista Ordenada (com números)

```html
<ol>
    <li>Primeiro</li>
    <li>Segundo</li>
    <li>Terceiro</li>
</ol>
```

### Lista Aninhada

```html
<ul>
    <li>Frutas
        <ul>
            <li>Maçã</li>
            <li>Banana</li>
        </ul>
    </li>
    <li>Legumes</li>
</ul>
```

## Links

### Link Básico

```html
<a href="https://google.com">Google</a>
```

### Link em Nova Aba

```html
<a href="https://google.com" target="_blank">Google (nova aba)</a>
```

### Link com Âncora

```html
<a href="#secao1">Ir para Seção 1</a>

<!-- Mais conteúdo -->

<h2 id="secao1">Seção 1</h2>
```

### Link para Email

```html
<a href="mailto:contato@email.com">Enviar email</a>
```

## Exercício

Crie uma lista não-ordenada (`<ul>`) com 3 itens (`<li>`):
1. HTML
2. CSS
3. JavaScript
