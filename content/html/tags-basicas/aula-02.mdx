## Introdução

Listas organizam itens em sequência. Links conectam páginas. Juntos, formam a base da navegação web.

## Conceito Explicado

### Listas

```html
<!-- Lista não-ordenada (com bullets) -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<!-- Lista ordenada (com números) -->
<ol>
    <li>Primeiro</li>
    <li>Segundo</li>
</ol>
```

### Links

```html
<a href="https://google.com">Google</a>
<a href="/sobre">Sobre nós</a>
<a href="mailto:contato@email.com">Enviar email</a>
```

## Exemplo Visual

**Lista `<ul>` vs `<ol>`:**
```html
<ul>
    <li>Maçã</li>
    <li>Banana</li>
</ul>
<!-- Resultado: • Maçã • Banana -->

<ol>
    <li>Maçã</li>
    <li>Banana</li>
</ol>
<!-- Resultado: 1. Maçã 2. Banana -->
```

**Menu de navegação:**
```html
<nav>
    <ul>
        <li><a href="/">Início</a></li>
        <li><a href="/sobre">Sobre</a></li>
        <li><a href="/contato">Contato</a></li>
    </ul>
</nav>
```

## Boas Práticas

- ✅ Use `<ul>` para menus (semântico)
- ✅ Use `<nav>` para navegação principal
- ✅ Links internos use caminhos relativos (`/sobre`)
- ❌ Não use `<br>` para criar listas
- ❌ Não coloque `<a>` dentro de outro `<a>`

## Atividade Final

Crie um menu de navegação com:
1. Uma lista `<ul>` com 3 itens
2. Cada item `<li>` contendo um link `<a>`
3. Links apontando para `/`, `/sobre`, `/contato`
