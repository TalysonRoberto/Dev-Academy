## Introdução

Flexbox é o sistema de layout mais usado no CSS moderno. Ele facilita alinhar e distribuir elementos.

## Conceito Explicado

```css
.container {
  display: flex;
  justify-content: space-between; /* Alinhamento horizontal */
  align-items: center;            /* Alinhamento vertical */
  gap: 16px;                      /* Espaço entre itens */
}
```

### Propriedades do Container
```css
flex-direction: row;    /* Horizontal (padrão) */
flex-direction: column; /* Vertical */
flex-wrap: wrap;        /* Quebra de linha */
```

### Propriedades dos Itens
```css
flex-grow: 1;   /* Cresce para preencher espaço */
flex-shrink: 0; /* Não encolhe */
flex-basis: 200px; /* Tamanho base */
```

## Exemplo Visual

```css
nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 16px;
}

nav ul {
  display: flex;
  gap: 16px;
  list-style: none;
}
```

**Preview:** Logo à esquerda, links à direita, alinhados verticalmente.

## Boas Práticas

- ✅ Use Flexbox para layouts unidimensionais (linha ou coluna)
- ✅ Use `gap` em vez de `margin` entre itens
- ✅ Use `flex-wrap: wrap` para responsividade
- ❌ Não use Flexbox para layouts 2D complexos (use Grid)

## Atividade Final

Monte a navbar:
1. `display: flex` no `<nav>`
2. `justify-content: space-between` para separar logo e links
3. `align-items: center` para alinhar verticalmente
