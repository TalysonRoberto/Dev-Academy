## Introdução

Seletores CSS determinam **quais elementos** serão estilizados. Existem 3 tipos principais.

## Conceito Explicado

### Seletor de Elemento
```css
p { color: blue; }        /* Todos os <p> */
h1 { font-size: 32px; }   /* Todos os <h1> */
```

### Seletor de Classe
```css
.destaque { color: green; }  /* Elementos com class="destaque" */
```

### Seletor de ID
```css
#titulo { font-weight: bold; }  /* Elemento com id="titulo" */
```

## Exemplo Visual

```html
<style>
  p { color: gray; }           /* Todos os parágrafos */
  .destaque { color: green; }  /* Apenas class="destaque" */
  #titulo { color: blue; }     /* Apenas id="titulo" */
</style>

<h1 id="titulo">Título Azul</h1>
<p>Parágrafo Cinza</p>
<p class="destaque">Parágrafo Verde</p>
```

**Preview:** Cada elemento recebe cor diferente baseado no seletor usado.

## Boas Práticas

- ✅ Use classes para estilos reutilizáveis
- ✅ Use IDs para elementos únicos
- ✅ Prefira classes sobre IDs na maioria dos casos
- ❌ Não use IDs para estilos que precisam ser reutilizados

## Atividade Final

Estilize os 3 elementos usando:
1. Seletor de elemento `p` → cor cinza
2. Seletor de classe `.destaque` → cor verde
3. Seletor de id `#titulo` → cor azul
