## Introdução

CSS (Cascading Style Sheets) controla a **aparência** visual do HTML. Sem CSS, todas as páginas seriam texto preto em fundo branco.

## Conceito Explicado

Existem 3 formas de aplicar CSS:

### 1. Inline (direto no elemento)
```html
<p style="color: red; font-size: 20px;">Texto vermelho</p>
```

### 2. Interno (tag `<style>`)
```html
<head>
  <style>
    p { color: red; font-size: 20px; }
  </style>
</head>
```

### 3. Externo (arquivo `.css`)
```html
<head>
  <link rel="stylesheet" href="estilos.css">
</head>
```

## Exemplo Visual

**Sem CSS:**
```html
<h1>Título</h1>
<p>Parágrafo</p>
```
→ Texto preto, tamanhos padrão do navegador.

**Com CSS:**
```css
h1 { color: blue; font-size: 32px; }
p { color: gray; line-height: 1.6; }
```
→ Título azul grande, parágrafo cinja com espaçamento.

## Boas Práticas

- ✅ Use CSS externo para projetos reais
- ✅ Use CSS interno para protótipos rápidos
- ✅ Use inline apenas para overrides pontuais
- ❌ Não misture estilos inline com externos desnecessariamente
