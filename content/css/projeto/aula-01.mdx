## Introdução

Este é o projeto final de CSS. Você vai aplicar todos os conceitos aprendidos para estilizar a landing page criada no projeto de HTML.

## Conceito Explicado

Combine todos os conceitos:

```css
/* Reset e base */
* { margin: 0; padding: 0; box-sizing: border-box; }

/* Layout com Flexbox/Grid */
nav { display: flex; justify-content: space-between; }
.hero { display: grid; place-items: center; min-height: 80vh; }

/* Responsividade */
@media (max-width: 768px) {
  nav ul { display: none; }
}

/* Efeitos */
.btn { transition: all 0.2s ease; }
.btn:hover { transform: scale(1.05); }
```

## Exemplo Visual

**Antes (HTML puro):**
- Texto preto, sem layout, sem cores

**Depois (com CSS):**
- Header fixo com navegação
- Hero com gradiente e CTA
- Cards com hover effects
- Responsivo para mobile

## Boas Práticas

- ✅ Comece pelo mobile (mobile-first)
- ✅ Use variáveis CSS para cores
- ✅ Teste em diferentes tamanhos de tela
- ❌ Não sobrescreva estilos desnecessariamente

## Atividade Final

Estilize a landing page:
1. Defina cores e tipografia
2. Crie layout com Flexbox/Grid
3. Adicione responsividade com media queries
4. Adicione efeitos hover e transições
